# Publishing-ready-graphs-EIS

This project focuses on generating publication-quality plots from Electrochemical Impedance Spectroscopy (EIS) data.

## Features
- Modular Python code
- Suitable for scientific papers and reports

## Project Structure
`data/` – raw and processed EIS data
`notebooks/` – analysis and visualization
`src/` – reusable Python functions
`figures/` – generated plots
